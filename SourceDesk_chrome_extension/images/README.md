# images/

Place the following PNG icon files in this directory before loading the extension:

| File           | Size       |
|----------------|------------|
| `icon-16.png`  | 16 × 16 px |
| `icon-32.png`  | 32 × 32 px |
| `icon-48.png`  | 48 × 48 px |
| `icon-128.png` | 128 × 128 px |

## Quick way to generate them

Open `../generate-icons.html` in any browser. It will draw the SourceDesk "S"
icon at all four sizes using `<canvas>` and offer download links for each PNG.
Save each file into this directory with the exact names listed above.
